# newmovie

How to use


```npm install```

create a new .env file and populate it with

DB_PASS=YOURMYSQL/MARIADB_PASSWORD

DB_USER=YOUR_MYSQL/MARIADB_USERNAME

DB_NAME=YOUR_DATABASE_NAME


```npm run dev```

Basic method

To get all user

```GET /user```

To get user by id

```GET /user/id```

To update user by id

```PATCH /user/edit/id```

To delete user by id

```DELETE /user/delete/id```
